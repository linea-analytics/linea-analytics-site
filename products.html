LINEA

LINEA is an R-library designed as an interface to accelerate linear regression modelling and visualization. The library allows users to build complex model specification through custom transformations.

Install it from CRAN.

Fork it on GitHub.

Learn it at linea-r.org.

LINEAPP

LINEAPP is a webapp based on the LINEA R-library designed as an interface to accelerate linear regression modelling and visualization. The library allows users to build complex model specification through custom transformations.

The app is hosted on Google Cloud in a secured instance, which provides security, stability, and simplicity to LINEAPP.
LINEAPP also contains exclusive features, not available in the R-Library. These are mainly Optimisations and Bayesian Regression.
